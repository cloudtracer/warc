# warc

Parse WARC (Web Archive Files) as a node.js stream

[![build status](https://secure.travis-ci.org/eugeneware/warc.png)](http://travis-ci.org/eugeneware/warc)

## Installation

This module is installed via npm:

``` bash
$ npm install warc
```

## Example Usage

``` js
var warc = require('warc');
```
